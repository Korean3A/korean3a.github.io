# 개인정보처리방침 작성 및 적용 계획

`privacy.html` 파일을 석면추방국민연대의 디자인 가이드라인에 맞춰 최신 개인정보처리방침 내용으로 업데이트합니다.

## 1. 개요
사용자가 제공한 텍스트와 스크린샷(Notion 스타일)을 기반으로, 전문적이고 시각적으로 깔끔한 개인정보처리방침 페이지를 구현합니다.

## 2. 주요 변경 사항
- **구조 설계**: 다른 페이지(`about.html`, `contact.html` 등)와 동일한 레이아웃(Nav, Footer 포함)을 유지하면서 본문 내용은 Notion의 깔끔한 디자인을 차용합니다.
- **콘텐츠 구성**:
    - **우리는 누구인가요**: 웹사이트 주소 명시
    - **댓글 및 미디어**: 수집 데이터 정의 및 EXIF 관련 주의사항
    - **쿠키**: 브라우저별 설정 방법 및 쿠키 보존 기간 상세 안내
    - **데이터 보존 및 권리**: 사용자 데이터의 보유 기간 및 권리(열람, 삭제 등) 안내
- **디자인 요소**:
    - `page-container` 클래스를 활용하여 가독성 높은 레이아웃 구성
    - `.notion-aside` 스타일을 사용하여 주요 안내 사항 강조
    - 반응형 디자인 적용 (모바일 최적화)

## 3. 기술 스택
- HTML5 / CSS3 (Vanilla CSS)
- 공통 컴포넌트(`nav.html`, `footer.html`) 로더 (`script.js`)

## 4. 작업 순서
1. `privacy.html` 파일에 기본 HTML 구조 작성 (Head, Body, Nav, Main, Footer).
2. 제공된 텍스트를 논리적인 섹션으로 나누어 본문에 배치.
3. CSS 변수 및 클래스를 활용하여 디자인 폴리싱.
4. `includeHTML` 및 외부 스크립트(ChannelIO 등) 연결.
